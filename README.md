## About
Junit-spockblocks is simple junit extension which allow us to use a some subset of blocks known
from a Spock framework:
 - given
 - and
 - when
 - then
 - expect

## Stack technology
 - junit 5
 - junit platform
 - asm library (bytecode manipulation and analysis framework)
